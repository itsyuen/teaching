# Courses to appear
This directory is meant to keep track of yearly courses, as well as weekly
meetings teaching done at any time with computer assitance.

## directory may used for single courses
Using .md files to link the many different direcories/files for updated
teaching notes, not necessary to re-write all the scripts here, just reuse
all the modules that exists.

## standard example for teaching courses and it's realtionship with modules

### relations
```text
teaching/
├── modules/
│   ├── cpp_basics/
│   ├── matlab_intro/
│   ├── ros2_topics/
│   └── simulink_control/
│
└── courses/
    ├── 2026_spring_intro_programming/
    ├── 2026_fall_ros2_mechatronics/
    ├── 2026_highschool_summer_camp/
    └── special_workshops/
```
### course structure with only .md(s)
```text
2026_fall_ros2_mechatronics/
├── README.md
├── syllabus.md
├── weekly_plan.md
├── grading.md
├── resources.md
└── links_to_modules.md
```
