# C++ Foundations: Namespace, Scope, Classes, and References
This teaching module is designed for beginner engineering students to build strong foundations in modern C++ programming.

## Learning Objectives

Students will practice and understand:

- namespaces and name conflicts
- variable scope (local, global, block scope)
- class creation using examples (`Person`, `Fruit`)
- constructors and member functions
- header/source file separation (`.h` + `.cpp`)
- vectors and arrays
- call by value
- call by reference
- call by pointer (address)
- use of `auto`
- basic enum usage
- multi-file compilation
- clean project structure

## Folder Structure

```text
fruit_class/
person1/
person2/
main1.cpp
myMath.cpp
myMath.h
```

## Suggested Compile Commands
g++ main1.cpp myMath.cpp -o app
./app
