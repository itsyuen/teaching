# MATLAB Intro to Numerical Simulation

## Purpose

Introductory module for engineering students to learn:

- MATLAB basics
- Variables and scripts
- Plotting
- Importing CSV data
- Numerical simulation concepts
- Simulink block-diagram modeling

## Files

- `intro_live_script.mlx` – guided classroom live script
- `intro_simulink_model.slx` – simulation example
- `sample_data.csv` – imported experimental/sample dataset

## Suggested Teaching Flow

1. Control model discrete equation derivation  
2. difference between theoretical numerical fitting and step wise time update  
3. Discrete matrix and continous matrix  
4. laplace transform for continous block system 
5. Compare analytical vs numerical thinking  
6. Open Simulink model  
7. Modify parameters and rerun

## Notes

Suitable for first-year or second-year engineering students.
